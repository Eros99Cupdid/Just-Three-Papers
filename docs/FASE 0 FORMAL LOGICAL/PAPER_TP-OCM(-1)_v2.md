## BAB II: MEKANIKA TP-OCM (TOPOLOGICAL PHASE-LOCKED MANIFOLD)

### 2.1 Kertas Manifold (The Constraint)
Partikel tidak bergerak bebas di ruang hampa (*Void*). Ia terikat pada permukaan geometri tak terlihat ("Kertas") yang ditentukan oleh Hukum Tangensial:
$$\tan(\theta_1) = \tan(\theta_2) \cdot \tan(\theta_3)$$
*Hukum ini menjaga partikel agar tetap berada dalam koherensi sistem, mencegahnya terlempar ke singularitas.*

### 2.2 Rotasi Kosmik & Phase Surfing
Sistem memperhitungkan rotasi inheren alam semesta sebesar $\Omega \approx 2.47 \times 10^{-3}$ rad/s.
* **Phase Surfing:** Teknik navigasi yang menggunakan pola gelombang sinusoidal untuk menumpangi arus magnetik. Ini mengurangi gesekan (Delta-V) dibandingkan menabrak "angin" secara frontal.

## BAB III: STRATEGI KELUAR (THE EXODUS PROTOCOL)

### 3.1 Pintu Belakang (The Heliotail Route)
Menembus heliosfer lewat depan (*Nose Shock*) memboroskan energi untuk melawan dinding radiasi.
* **Strategi:** Keluar melalui "Ekor" (*Heliotail*).
* **Keuntungan:** Arus angin surya searah, perlindungan magnetik lebih panjang, turbulensi minimal.

### 3.2 Sudut Emas (The 45-Degree Cut)
Jangan keluar tegak lurus (90°) yang mematikan momentum, dan jangan lurus (0°) yang kotor oleh debu.



* **Solusi:** Gunakan sudut inklinasi **±45 derajat**.
* **Fisika:** Titik keseimbangan terbaik (*Sweet Spot*) antara menjaga momentum orbital (bantuan kecepatan rotasi) dan keluar dari bidang ekliptika yang kotor.

import numpy as np
import pandas as pd

def tpocm_vector_correction(target_pos, time_delay, omega=2.47e-3):
    """
    Menghitung posisi target terkoreksi (Dynamic Targeting)
    akibat Cosmic Spin.
    """
    theta = omega * time_delay
    # Matriks Rotasi Z-axis
    R = np.array([
        [np.cos(theta), -np.sin(theta), 0],
        [np.sin(theta),  np.cos(theta), 0],
        [0,              0,             1]
    ])
    return np.dot(R, target_pos)

def analyze_exit_angle_efficiency():
    """
    Membandingkan efisiensi energi untuk berbagai sudut keluar.
    Efficiency = Momentum Retention (Cos) * Clearance Speed (Sin)
    """
    angles = [0, 30, 45, 60, 90]
    results = []
    
    for deg in angles:
        rad = np.radians(deg)
        # Momentum orbital yang bisa 'dicuri' (Cos)
        momentum = np.cos(rad)
        # Kecepatan menjauh dari bidang kotor (Sin)
        clearance = np.sin(rad)
        # Skor Gabungan (Harmonic Mean proxy)
        score = momentum * clearance 
        
        results.append({
            "Sudut": deg,
            "Momentum": round(momentum, 3),
            "Clearance": round(clearance, 3),
            "Efisiensi Total": round(score, 3)
        })
    
    return pd.DataFrame(results)

# Output Simulasi
# Sudut 45 derajat akan menghasilkan skor 'Efisiensi Total' tertinggi (0.500)
# Dibandingkan 30 derajat (0.433) atau 60 derajat (0.433).
